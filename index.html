Introdução

Python é uma linguagem de programação interpretada de alto nível e propósito geral.

Principais características:

  • Sintaxe clara e legível
  • Tipagem dinâmica
  • Multi-paradigma
  • Grande comunidade de desenvolvedores
print("Olá Mundo!")
Sintaxe Básica

Exemplo de declaração de variáveis:

nome = "João" idade = 30 altura = 1.75

Blocos são definidos por indentação:

if idade >= 18: print("Maior de idade") else: print("Menor de idade")
Estruturas de Controle

Estruturas condicionais:

nota = 8.5 if nota >= 7: status = "Aprovado" elif nota >= 5: status = "Recuperação" else: status = "Reprovado"

Loops iterativos:

  • For loop
  • While loop
  • Compreensões de lista
Funções

Definição básica de funções:

def calcular_imc(peso, altura): return peso / (altura ** 2)

Parâmetros opcionais:

def saudacao(nome="usuário"): print(f"Bem-vindo, {nome}!")
Listas e Dicionários

Exemplo de lista:

frutas = ["maçã", "banana", "laranja"] numeros = [1, 2, 3, 4, 5]

Exemplo de dicionário:

pessoa = { "nome": "Maria", "idade": 25, "cidade": "São Paulo" }
  • Listas são ordenadas e mutáveis
  • Dicionários armazenam pares chave-valor
